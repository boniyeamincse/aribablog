# Abriba - Modern Laravel Blog Platform

Abriba is a high-performance, modular blog engine built with Laravel 12. It features a scalable architecture designed for high content volume, advanced SEO, and seamless user interaction.

## Core Features

- **Modular Architecture**: Built with separation of concerns in mind.
- **Advanced Blogging**: Support for rich media, categories, and tags.
- **Robust Security**: Built-in protection against common web vulnerabilities.
- **Full API Support**: Headless ready with a complete RESTful API suite.
- **Engagement Tools**: Integrated comments and notification systems.

## Tech Stack

- **Backend**: Laravel 12, PHP 8.2+
- **Frontend**: Blade, Vite, Vanilla JS/CSS (Premium UI)
- **Database**: MySQL / PostgreSQL
- **Cache**: Redis
- **Security**: Laravel Sanctum (API Auth)
